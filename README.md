# JsonToCSV
Simple program to convert JSON files to CSV.<br>
It can convert simple JSON files to CSV.

## SAMPLE
```JSON
[
    {
    "Id": 1,
    "Name": "Mark"
    },
    {
    "Id": 2,
    "Name": "Tom"
    },
    {
    "Id": 3,
    "Name": "Bill"
    },
    {
    "Id": 4,
    "Name": "John"
    }
]
```
<br><br>
It produce this:<br>
Id;Name<br>
1;Mark<br>
2;Tom<br>
3;Bill<br>
4;John<br>
